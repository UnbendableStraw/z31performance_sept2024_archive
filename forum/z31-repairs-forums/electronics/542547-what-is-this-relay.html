    what have you swapped and/or have done so far. if the fuel pump primes when the key is turned on the ecu is powering up. if the ecu has no input from the crank sensor in the distributor it will not attempt to fire the injectors or send a spark.

    the fuel injectors do have power all the time even with the key off. the ecu triggers the ground side switching to fire them.

    have you tried getting the codes from the ecu? if you get a code 11 then chances are you are not getting a engine speed.

    code 13 is for a cylinder head temperature sensor. if its not working and cold out its almost impossible to start with out it. the ecu goes into a programmed curve mode and assumes the temperature is 68 degrees out.

    code 23,24,31 can be ignored for now as they require input from the user to clear them.

    are you getting spark and injector pulse's?
