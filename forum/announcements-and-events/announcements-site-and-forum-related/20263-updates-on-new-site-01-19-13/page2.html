    Re: Updates on New Site - 01/01/13

    consider increasing the php memory size from default 64 to something like 256 at least. VB has imagecache functions that are going to cause a lot of time outs and memory errors when people upload pictures that are resized and cached into a temp folder on the server. you might want to also configure the php timeout length as well.

    are you using an external database manager, or are you using the phpmyadmin applet on the webserver? i would upload using an external application. it's much quicker and doesn't time out at all.

    also, does IMPEX do database to database on localhost? or does it require an installed database and an uploaded file to do its import/exporting?

    if it could be done over localhost connection, it would be super fast.
