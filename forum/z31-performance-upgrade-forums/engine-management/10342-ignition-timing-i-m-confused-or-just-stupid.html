    Re: Ignition timing (I'm confused or just stupid)

    ECU sees:
    rpms from CAS (crank angle sensor inside distributor)
    AFM voltage readings
    CHTS voltage readings
    Knock Sensor voltage (if equiped)
    and WOT switch from inside the TPS.

    Then the ECU determines how much ignition timing to adjust from from static timing (physically set timing with timing light).

    If you you set the static timing at 15* when it calls for 20*, then the ECU will still adjust timing that it was preprogrammed for based on sensor inputs, but all timings will be 5* less since you set the physical timing at 15*.
