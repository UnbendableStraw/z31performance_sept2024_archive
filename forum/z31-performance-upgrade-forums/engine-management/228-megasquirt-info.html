    It's very easy. You can use the stock ignitor and coil and such.

    If you already have the CAS hacked to feed MS the tach signal, then all you need to do is select the FIDLE output as a spark output (or, I believe the v3 board and updated software allows a seperate dedicated spark output pin so you can save you fast idle output). You hook that to the base of the ignitor…

    In the (1984) FSM, the ignitor is labeled as the "Power Transistor", and it is the green wire with a black stripe (G/B) going to the transistor next to the coil.

    Just hook that up to the output from MS, however, you will need to hook up a +5v pullup resistor to basically amplify the signal from MS enough to be able to trigger the coil.

    Here is a diagram from my site: http://blozup.com/ms.php#vehiclewiring
