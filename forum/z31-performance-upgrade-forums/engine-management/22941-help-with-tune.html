    Have you adjusted the cold start enrichment. Warm start enrichment. After idle enrich? All of these "tables" need to be tweaked after changing injector size. They're just a percentage to enrichen the mixture based on engine temp..the ECU uses them depending on which start it is etc. You'll be fighting idle AFRs until warmed up forever without tweaking those.
