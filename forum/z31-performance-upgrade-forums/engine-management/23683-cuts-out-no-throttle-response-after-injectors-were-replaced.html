    cuts out, no throttle response after injectors were replaced

    04-27-2014, 10:13 AM
    replaced injectors a week ago, and replaced vacuum lines/fuel lines. Fixed a few vacuum leaks etc. Now, starting it up for the first time, i had a pretty good backfire. Afterwards, it would idle like crap and have poor throttle response. If i pinched off the return hose, it would run decent or If i unplug the MAF, it would run normally for the most part part. So i swapped in another MAF, and ran fine, for about a day. Now its returned with same issue. I can't even start it up barely, it just wants to idle at like 400-500 rpms. If i pinch off my return hose, it'll run better, and idle normally.

    On start-up it may rev up to 1400 rpms (cold), but then a few seconds later it falls on its face, it sounds like i'm losing like 3 cylinders, gets very lopey. Heres what i got so far:

    Timing: 15*
    fuel pressure: 34psi at idle, 42 with regulator unhooked, and 60psi with return pinched off
    no vacuum leaks found, i've sprayed right on injector/intake seals
    no codes stored in the ECM
    vacuum gauge is showing 10" when its running like garbage, and then around 20" when i pinch off the return.
    Plug wires are hooked up correctly
    If MAF is unplugged, will run 80-90% better, still has some slight hesitation (could be because of MAF fail-safe though)

    Things of interest:
    Removing upper plenum, the EGR tube developed a crack where it screws into exhaust manifold, so now i have an exhaust leak
    I had the distributor upside down for a day or 2 when i was doing valve cover gaskets/refinishing (maybe CAS got oil or some shit in it?)
    It ran good enough to drive before i fixed all the vacuum leaks and injectors

    Yesterday, after i swapped in the MAF it ran fine for about 45min, on my way home from my friends. Had power, no missing, had the exhaust leak even then, and it ran fine, just sounded like crap. IT wasn't until i went to get some food, that it acted up in the drive thru.

    FIXED!

    Had a bad EGR solenoid. So when i redid all the vacuum lines correctly, it revealed itself. The PO had looped the vacuum lines at the throttle body. Once i routed them right, it fed vacuum into the EGR solenoid (which was stuck partially open), opening up the EGR and screwing up my idle. So i just took the vacuum line off at the throttle body, popped a vacuum plug on it, and its good now.
