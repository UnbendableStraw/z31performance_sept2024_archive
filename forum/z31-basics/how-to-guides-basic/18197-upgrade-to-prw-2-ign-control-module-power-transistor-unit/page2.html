    Re: Upgrade to PRW-2 Ign. Control Module (Power Transistor U

    did some searching, looks like the PRW-2 transistor is currently Nissan part # 22020-56E16

    list of cars it's found on: http://www.nissanpartszone.com/Page_Pro … 2020-56E16

    to answer a question, for the Sentra, it's for the GA16 only. for the D21, VG only.

    the other PTU is part # 22020-53J20.

    used to be 22020-56E11 for all of these models (what it states here and on lookup at any dealer's TradeMotion catalog): http://www.nissanpartszone.com/Page_Pro … 2020-56E11

    22020-53J20 also fits 91-94 SR20 powered vehicles (Sentra/NX/G20).
